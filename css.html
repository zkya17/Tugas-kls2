Jurusan RPL

Profil Jurusan PPLG

Program Pengembangan Perangkat Lunak dan GIM atau sebelumnya keahlian Rekayasa Perangkat Lunak (RPL) adalah salah satu kompetisi keahlian dalam bidang Teknologi Komputer dan Informatika yang secara khusus mempelajari tentang pemrograman komputer. Lulusan RPL dewasa ini sangat dibutuhkan untuk menjawab banyaknya kebutuhan industri bagi ketersediaan tenaga Teknisi dalam bidang pengmbangan software. Beberapa ruang lingkup pekerjaan di bidang RPL adalah Software Developer, Game Developer, Software Tester, Software Abalisis dan Integrator Konsultan IT dan Progammer

Peluang karir jurusan RPL bisa melingkupi software engineer/progammer, monile computing developer, IT consultant, System analyst, Game developer software tester dan masih banyak lagi prospek karir lainnya
